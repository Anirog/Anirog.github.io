## Basic Syntax

Heading
# H1  
## H2  
### H3

Bold
**Bold Text**

Italic
*italicized text*

Blockquote
> blockquote

Ordered List
1. First item
2. Second item
3. Third item

Unordered List
- First item
- Second item 
- Third item

Code
`code`

Horizontal Rule
---

Link
[title](https://www.example.com)

Image
![alt text](https://placehold.co/720x1080?text=Image)

## Extended Syntax

Fenced Code Block
```  
	h1 {
		border: 1px solid red;
	}
```
